# Pasantías en Investigación Económica

## Sistema de Estandarización VENCOLECO

Durante mis pasantías, desarrollé un **sistema completo de estandarización** para un proyecto de investigación de gran escala sobre la dinámica económica entre Venezuela y Colombia (2013-2025).

### 📊 Alcance del Proyecto

- **13 estudiantes** de Economía Internacional
- **13 años** de análisis (2013-2025)
- **3 dimensiones** de investigación: balanza comercial, flujos migratorios, marco normativo
- **4 fases** de ejecución con metas quincenales

### 🎯 Objetivos del Sistema

1. **Estandarizar** todos los entregables bajo normas APA 7ma edición
2. **Coordinar** la recolección de datos entre 13 investigadores
3. **Garantizar** la calidad mediante checklists y revisiones entre pares
4. **Documentar** metodologías de búsqueda y limpieza de bases de datos

### 📚 Entregables Desarrollados

| Documento | Descripción |
|-----------|-------------|
| Manual APA 7ma Edición | Guía completa de referencias bibliográficas, citas textuales, formato de tablas |
| Plan de Trabajo Integral | Cronograma de 4 fases, asignación de años, sistema de nomenclatura |
| Checklists de Calidad | Validación por entregable: BD comercial, fichas normativas, capítulos |
| Matriz de Evaluación | Sistema de 100 puntos para revisión entre pares |
| Plantillas Estandarizadas | Excel para datos comerciales, Word para fichas, Markdown para resúmenes |

### 🔍 Variables de Investigación

**Dimensión 1 - Balanza Comercial:**
- Importaciones/exportaciones bilaterales (USD)
- Análisis sectorial (códigos CIIU/HS)
- Evolución mensual/trimestral
- Productos principales y diversificación

**Dimensión 2 - Flujos Migratorios:**
- Estadísticas de entradas/salidas
- Caracterización poblacional
- Políticas migratorias aplicadas
- Impacto en mercado laboral

**Dimensión 3 - Marco Normativo:**
- Acuerdos comerciales firmados/modificados
- Decretos y resoluciones presidenciales
- Tratados bilaterales de inversión
- Decisiones de organismos de integración

### 🛠️ Herramientas Utilizadas

- **Bases de datos:** BCV, DANE, BanRep, UN Comtrade, CEPALSTAT
- **Software:** Excel avanzado, SPSS/Stata para análisis
- **Gestión:** Sistema de nomenclatura VENCOLECO_[TIPO]_[SUBTIPO]_[PERÍODO]_[VERSIÓN]_[AUTOR]
- **Metodología:** APA 7ma edición, revisión entre pares, control de calidad

### 📊 Sistema de Calidad

Implementé un protocolo de revisión con:
- **4 grupos** de supervisión según complejidad de años asignados
- **Revisiones semanales** individuales
- **Revisiones quincenales** grupales
- **Evaluación entre pares** con matriz de 100 puntos

**Escala de Calificación:**
- 88-100: Excelente (aprobado sin correcciones)
- 68-88: Bueno (correcciones menores)
- 60-67: Aceptable (correcciones moderadas)
- 51-59: Regular (correcciones sustanciales)
- 0-50: Deficiente (debe rehacerse)

### 📖 Capacitaciones Diseñadas

**Normas APA 7ma Edición (4 sesiones):**
1. Citas textuales y paráfrasis
2. Referencias bibliográficas
3. Tablas y figuras
4. Revisión final y casos especiales

**Bases de Datos (2 sesiones):**
1. Fuentes oficiales y organismos multilaterales
2. Análisis con software (SPSS/Stata/Excel)
